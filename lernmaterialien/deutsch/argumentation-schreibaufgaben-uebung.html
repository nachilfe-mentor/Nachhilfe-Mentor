Stufe 1: Vorhersage prüfen

These: Betreute Lernzeiten können Hausaufgaben an zwei Tagen pro Woche sinnvoll ersetzen.

Beispiel: Wenn jemand eine Matheaufgabe nicht versteht, kann die Lehrkraft direkt eine Rückfrage klären.

Prüfen: Welche Satzkarte ist die passende Schlussregel?

Musterlösung zum Vergleichen

Betreute Lernzeiten könnten Hausaufgaben an zwei Tagen pro Woche sinnvoll ersetzen, weil Schülerinnen und Schüler bei Schwierigkeiten sofort Unterstützung erhalten. Wer zu Hause an einer Aufgabe hängenbleibt, übt oft nicht weiter, sondern rät oder gibt auf. In einer Lernzeit kann dagegen eine Lehrkraft eine Rückfrage klären, sodass die Übungszeit wirklich zum Verstehen beiträgt. Das Beispiel zeigt: Entscheidend ist nicht nur, dass Aufgaben erledigt werden, sondern ob sie zu sicherem Lernen führen. Zwar könnte man einwenden, dass Hausaufgaben Selbstständigkeit trainieren. Dieses Ziel bleibt wichtig, aber es wird nicht aufgegeben, wenn nur zwei Tage ersetzt werden und die übrigen Aufgaben weiterhin eigenständig bearbeitet werden. Deshalb wäre eine begrenzte Einführung von Lernzeiten ein sinnvoller Kompromiss.

Struktur-Scaffold

  1. Position: Ich vertrete die Ansicht, dass ...
  2. Begründung: Ein wichtiger Grund dafür ist ...
  3. Beispiel: Das zeigt sich zum Beispiel, wenn ...
  4. Schlussregel: Dieses Beispiel stützt die These, weil ...
  5. Gegenargument: Zwar spricht dagegen, dass ...
  6. Entkräftung: Dennoch überzeugt das nur teilweise, denn ...
  7. Fazit: Insgesamt ist deshalb ...

Bewertungsraster

Kriterium Stark Noch überarbeiten
Argumentationslogik These, Grund, Beispiel und Schlussregel bauen sichtbar aufeinander auf. Ein Beispiel steht im Text, aber seine Bedeutung für die These bleibt unklar.
Gegenargument Ein Einwand wird fair genannt und mit einer passenden Einschränkung entkräftet. Der Einwand fehlt oder wird so schwach dargestellt, dass die Entkräftung nicht überzeugt.
Sprache Verknüpfungen zeigen die Denkschritte: weil, daher, zwar, dennoch, insgesamt. Sätze stehen nebeneinander, ohne dass die logische Verbindung klar wird.

Typische Fehler und Korrektur

Fehler: Beispiel ohne Schlussregel

Korrektur: Ergänze einen Satz mit "Das zeigt, dass ...", der die Bedeutung des Beispiels erklärt.

Fehler: Begründung wiederholt die These

Korrektur: Frage dich, welcher neue Grund deine Position stützt. Der Grund darf nicht nur die Überschrift deines Textes wiederholen.

Fehler: Gegenargument wird abgewertet

Korrektur: Formuliere den Einwand so fair, dass jemand mit anderer Meinung ihn wiedererkennen würde. Entkräfte danach sachlich.
