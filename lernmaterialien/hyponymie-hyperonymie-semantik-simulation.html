Klasse 10–13 · Linguistik · Abitur

🏠 Was sind semantische Wortbeziehungen?

Wörter stehen in der Semantik in festen Beziehungen zueinander. Die wichtigsten sind:

Hyperonym (Oberbegriff) Ein Wort, das einen allgemeineren Begriff ausdrückt.
Tier ist Hyperonym von Hund.
Hyponym (Unterbegriff) Ein Wort, das einen spezielleren Begriff ausdrückt.
Hund ist Hyponym von Tier.
Meronym (Teilbegriff) Ein Wort, das einen Teil eines Ganzen bezeichnet.
Finger ist Meronym von Hand.
Synonymie (Bedeutungsähnlichkeit) Zwei Wörter mit gleicher oder sehr ähnlicher Bedeutung.
Sofa ≈ Couch
Prüffrage Hyponymie: „Ein X ist ein Y.“ → Wenn wahr: X = Hyponym von Y
Prüffrage Meronymie: „Ein X ist Teil von Y.“ → Wenn wahr: X = Meronym von Y
Beispiel: „Ein Pudel ist ein Hund.“ ✓ → Pudel ist Hyponym von Hund.
